# Deferred Features

- Advanced contractor detail visualizations beyond transaction-derived summary tables.
- Automatic recipient profile resolution when no stable recipient identifier is present.
- Agency-component configuration candidates for additional agencies. Production behavior remains deterministic and registry-based.
- Export/download workflows and broad trend analysis from the parent dashboard.

The solicitation extraction workflow, AI mapping, package recovery, diagnostics, and document processing are intentionally removed rather than deferred for this product.
